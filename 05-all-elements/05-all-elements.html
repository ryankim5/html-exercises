All elements

<a>

The anchor element (<a>) is one of the fundamental elements in HTML.

It allows a user to click on the link text it provides, and navigates to that link's location.

<abbr>

The abbr element (<abbr>) is one of the fundamental elements in HTML.

If you hover over the text that the <abbr> has, a little textbox will appear, showing the definition of the abbreviation.

<p>

The paragraph element (<p>) is one of the fundamental elements in HTML.

It allows the HTML file to have a paragraph of text.

<li>

The list item element (<li>) is one of the fundamental elements in HTML.

It allows to add a list item to lists. (e.g. the <ul> list)

<ul>

The unorderd list element (<ul>) is one of the fundamental elements in HTML.

It allows a list to be unordered.

<div>

The div element (<div>) is one of the fundamental elements in HTML.

It allows you to add a class to the <div>, so you can style it with CSS.

<titie>

The title element (<title>) is one of the fundamental elements in HTML.

The title lets you to choose what Tab name to use if the HTML file is opened in a web browser.

<h1> to <h6>

The element(s) <h1> to <6> are one of the fundamental elements in HTML.

It lets you to add a heading, from size 1 (biggest) to size 6 (smallest).

<img>

The element <img> is one of the fundamental elements in HTML.

It lets you to add a image to your HTML site.

THIS TAG (<img>) NEEDS A SRC (SOURCE OF THE IMAGE) AND DOESN'T NEED A CLOSING TAG.
<small>

The element <small> is one of the fundamental elements in HTML.

It lets you too add smaller text.
